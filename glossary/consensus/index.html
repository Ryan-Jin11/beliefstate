Belief State Field Guide Glossary

Consensus

Definition: Consensus is the state in which multiple credible sources converge on a shared explanation, allowing beliefs to be repeated with confidence.

What this is

  • A stabalizing force for belief states.
  • A prerequisite for confident synthesis.

What this is not

  • No popularity.
  • Not unanimity.

Example signals

  • Signal: Repeated phrasing across sources.
  • Signal: Consistent explanations with minor variation.
